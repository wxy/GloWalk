# GloWalk UI 设计规范

> 随行路灯 · GloWalk · 视觉设计系统
> 适用平台：iOS 15+ · 强制暗色模式 · SwiftUI 实现

---

## 1. 视觉基调

GloWalk 的核心隐喻不是"一个 App 界面"，而是**手中握着一盏暖光灯笼**。屏幕不是屏幕——是光源。所有视觉决策从这个隐喻出发。

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **光是主角，文字是配角** | 中央灯笼动画是视觉重心，所有信息元素围绕它，不喧宾夺主 |
| **消失比显示更有力量** | 好的暗面 UI 让用户忘记它的存在。琥珀色元素是暗示，不是宣告 |
| **动效只服务物理感** | 亮度变化是"渐亮/渐暗"不是"切变"。一切动效模拟物理世界光的行为 |
| **全局暗色** | 不跟随系统亮暗模式。强制暗色，从打开到关闭始终是"夜"的 |

### 1.2 参考系统

| 参考 | 借鉴 |
|------|------|
| Apple Watch 夜间模式 | 大面积纯黑 + 最小化信息 |
| 飞机驾驶舱 | 琥珀色照明保护暗视力的科学依据 |
| 天文馆红光照明 | 长波长不破坏视杆细胞（暗视力） |
| 手提灯笼 | 物理隐喻：手握光源，光随人走 |
| iOS 原相机手电筒 | 我们在取代和超越的东西 |

### 1.3 全局暗色模式

在 `Info.plist` 中设置 `UIUserInterfaceStyle = Dark`，强制整个应用暗色。不跟随系统亮暗模式切换。

## 2. 灯笼动画（核心视觉元素）

### 2.1 视角

**顶视图偏 10-15° 微倾角**——用户低头看手中的灯笼，从灯笼上口往下看。

```
          ┌─────┐
          │  ✦  │  ← 提手（最近、最大）
          │  ✦  │
       ╭──────────╮
       │  ✦✦✦✦✦  │  ← 灯笼上口（椭圆形，微倾角产生深度）
       │ ✦✦✦✦✦✦ │     可见灯笼内壁材质
       │  ✦✦✦✦✦  │
       ╰────┬─────╯
            │
     ═══════╧═══════  ← 地面光晕扩散（多层径向渐变）
```

- 不需要画完整的灯笼侧面——椭圆形上口 + 微妙的深度暗示足够
- 灯笼体用透光材质感（半透明琥珀色渐变叠加）而非几何线条
- 发光是主体，结构只是光的容器

### 2.2 动画规格

灯笼动画为**固定动画**，不响应手机运动。由设计师产出 Lottie 动画文件。

| 层级 | 内容 | 动画 |
|------|------|------|
| **Layer 1** | 提手 | 极微弱的上下浮动（模拟呼吸） |
| **Layer 2** | 灯笼体（上口椭圆形 + 内壁材质） | 光在腔内缓慢流转 |
| **Layer 3** | 粒子 | 1-2 个微光粒子从灯笼口缓慢上浮，3-5s 生命周期 |

技术实现：Lottie 文件 + SwiftUI 光晕叠加层（地面光晕由代码驱动以联动手电筒亮度）。

### 2.3 地面光晕（代码驱动）

3 层 `RadialGradient` 叠加，强度实时响应手电筒输出亮度：

```
  内层：半径 ~80pt，中心暖白 #FFF3E0，透明度 = 15% × brightness
  中层：半径 ~180pt，琥珀 #FFB74D，透明度 = 8% × brightness
  外层：半径 ~350pt，琥珀 #FFA726，透明度 = 3% × brightness → 边缘纯黑
```

光晕叠加 4-6 秒正弦呼吸波动（±5% 透明度），模拟火焰/灯光的不稳定感——不是闪烁，是极缓慢的呼吸。

## 3. 配色体系

### 3.1 琥珀色系（品牌色）

```
Amber-10   #FFF3E0   灯笼芯白热      仅用于光源中心
Amber-20   #FFE0B2   灯笼内壁亮部      
Amber-40   #FFCC80   灯笼内壁、发光元素  
Amber-60   #FFB74D   主琥珀色（品牌色） 大多数 UI 元素
Amber-80   #FFA726   琥珀色加深        
Amber-90   #FF8F00   琥珀色暗部        
Amber-Dim  #996A00   琥珀色禁用态      
```

### 3.2 暗色系（背景）

```
Black-0    #000000   纯黑             HUD 主背景（OLED 像素关闭）
Black-5    #0A0A0A   极暗灰           卡片背景、闪屏
Black-10   #1A1A1A   暗灰             设置页、历史页背景
Black-15   #262626   分割线/边框       
Black-20   #333333   暗色文字底色      
```

### 3.3 语义色

| 语义 | 暗面 | 亮面（如需要） | 规则 |
|------|------|------|------|
| 成功 | 不使用绿色（冷光破坏暗视力） | `#4CAF50` | 暗面用琥珀替代 |
| 警告 | `Amber-80` | `#FFA726` | 一致 |
| 错误 | Amber-60 闪烁 | `#EF5350` | 暗面不用红色 |
| 信息 | `Amber-60` | `Amber-60` | 一致 |

### 3.4 暗面不透明度层级

暗面 UI 没有"颜色深浅"——在纯黑背景上，用**透明度**建立信息层级：

```
  100%   主要交互元素（灯笼光晕、发光圆环）
  80%    重要状态信息（亮度百分比数字）
  60%    信息卡文字（月相、天气状态）
  40%    次级信息（距离、时间）
  25%    装饰/分隔元素
  15%    背景中的暗示（光晕中层）
  8%     地面光晕外层
  3%     地面光晕最外层
  0%     纯黑（安静区域、OLED 关闭像素）
```

## 4. 字体与排版

### 4.1 字体家族

| 用途 | 字体 | 权重 | 理由 |
|------|------|:---:|------|
| 暗面数字（亮度%） | SF Pro Rounded | Light / Regular | 圆润减少冷硬感 |
| 暗面信息（距离、时间） | SF Mono | Regular | 等宽便于扫读，"仪表读数"感 |
| 亮面/设置标题 | SF Pro | Semibold | iOS 原生风格 |
| 亮面/设置正文 | SF Pro | Regular | 标准阅读 |
| 中文 | 苹方 | — | 系统默认中文字体 |

### 4.2 字体大小阶梯

**暗面（所有尺寸比标准 UI 略小——屏幕离眼睛近 + 保护暗视力）：**

```
  28pt  Light      亮度百分比 "45%"
  14pt  Regular    发光区域下方标注
  12pt  Regular    信息卡文字（月相、天气）
  10pt  Regular    信息卡辅助文字
  12pt  Mono       底部状态栏（距离、电量、时间）
```

**亮面/设置页（标准 iOS 尺寸）：**

```
  34pt  Bold        页面标题
  22pt  Medium      闪屏宣传语短语
  17pt  Regular     设置项、列表行标题
  13pt  Regular     宣传语解释、辅助说明
  12pt  Regular     底部 tips、次要信息
```

### 4.3 排版规则

| 规则 | 说明 |
|------|------|
| 中文使用全角引号 `""` | 宣传语中不要出现英文引号 `""` |
| 暗面数值左对齐 | 图标在左，数值紧随；快速扫读时眼睛只需找图标位置 |
| 暗面间距宽松 | 比正常 UI 大 10-15%，低光下拥挤文字难以辨认 |
| 等宽数字用于变化值 | 距离、时间、电量变化时数字位置不跳动 |
| 暗面文字不用纯白 | 纯白在纯黑上对比度 21:1 刺眼。用 `white.opacity()` 或琥珀色 |

## 6. 动效规范

### 6.1 原则

动效只有一个目的：**让光的行为像物理世界的光**。不做纯装饰性动效。

### 6.2 动效清单

| 动画 | 触发条件 | 时长 | 曲线 | 说明 |
|------|---------|:---:|------|------|
| 闪屏→HUD 过渡 | 闪屏结束 | 1.0s | ease-in-out | 宣传语淡出 + 灯笼淡入 |
| 开灯 | 启动/恢复照明 | 0.5s | ease-out | 灯笼从暗到亮，光晕从小扩大 |
| 关灯 | 结束/双击 | 0.5s | ease-in | 灯笼渐暗→缩小→消失，模拟"吹灭" |
| 亮度自动渐变 | 自动调节触发 | 100ms/step | linear | 每步 10%，总时长取决于亮度差 |
| 信息卡切换 | 点击 🌙/☁️ | 0.2s | ease-out | 卡片半透明→变灰+删除线 |
| 粒子上升 | 持续循环 | 3-5s/个 | ease-out | opacity 0.6→0，ease-out |
| 光晕呼吸 | 持续循环 | 4-6s/周期 | sine | 地面光晕透明度 ±5% 正弦 |

### 6.3 暗面不动的东西

| 元素 | 理由 |
|------|------|
| 亮度数字 | 频繁跳动引起焦虑。值不变就不动，变化时 0.2s 淡入新值 |
| 底部状态栏 | 距离和时间不需要实时跳动动画 |
| 电池细线 | 变化极慢（分钟级），不需要动画 |

## 10. 实现备注

### 10.1 已确定的方案

| 决策 | 结论 |
|------|------|
| 全局模式 | 强制暗色，不跟随系统 |
| 灯笼动画 | Lottie 固定动画，不响应运动（由设计师后续产出） |
| 地面光晕 | SwiftUI `RadialGradient` 代码驱动，联动手电筒亮度 |
| 粒子效果 | Lottie 内包含，或 SwiftUI `Canvas` + `TimelineView` |
| 配色 | 全应用暗色系，琥珀色为唯一强调色 |
| 动效 | 仅物理隐喻驱动的动效，无装饰性动效 |

### 10.2 待设计师产出

| 素材 | 格式 | 规格 |
|------|------|------|
| 灯笼动画（含粒子） | Lottie `.json` | ~200×200pt 画布，暗色背景透明 |
| App 图标 | `1024×1024` PNG | 暗底 + 琥珀色灯笼标志 |
| 锁屏小组件图标 | SF Symbol 或自定义 | 简化版灯笼标志 |
